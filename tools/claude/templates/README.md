# tools/claude/templates/

Ready-to-copy starter files. Drop into `.claude/` in your own repo and customize.

## Contents

- `agents/` — agent definition templates (frontmatter, system prompt, deferral rules).
- `skills/` — skill (slash command) templates.
- `rules/` — operational rule file templates.
- `project/` — project-level docs that often accompany a `.claude/` directory (navigation hub, codebase guide, GitHub Actions guide, DevOps/infra guide).

## How to use

1. Copy the relevant template into your repo's `.claude/` tree.
2. Replace placeholders and `<TODO>` markers.
3. Cross-reference the matching guide in `../guides/` for the rationale behind each section.
