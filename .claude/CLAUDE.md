# CLAUDE.md

Promptcraft — open-source prompt engineering knowledge base and toolkit.

## Code Standards

- This is a public repository — all commits are permanent and visible
- No PII, company names, internal service names, or workspace/account IDs in any content
- Conventional commits: `feat:`, `fix:`, `docs:`, etc.
- Always work via PRs — never push directly to main

## Review Posture

- Scan every diff for PII/company data before committing
- Treat all content as public-facing — quality bar is external open-source level
- Multiple self-review passes before presenting changes
