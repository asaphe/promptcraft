# shared/languages/

Per-language coding standards. Tool-agnostic — apply whether you're using Claude Code, Cursor, ChatGPT, or any other assistant.

## Contents

- `general-standards.md` — cross-language conventions (formatting, tests, commits).
- `python.md` — Python style, typing, tooling.
- `typescript-javascript.md` — TS/JS style, strict mode, framework defaults.
- `bash.md` — shell scripting standards (`set -euo pipefail`, quoting, portability).
- `java-go.md` — Java and Go conventions.
